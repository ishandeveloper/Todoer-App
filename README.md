## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/Todoer-App.git
```
  ##### Move to the desired folder
```bash
cd \Todoer-App
```

  ##### To run the app, simply write
```bash
flutter run
```
